功能描述：<br>
1.具备中英翻译功能，若不输入任何字符直接按Enter则自动翻译剪切板内容<br>
2.中译英统一使用在线词典<br>
3.英译中，默认使用离线词典，若离线词典数据没有查找的单词则使用在线词典查询，然后添加该单词数据至离线词典<br>
